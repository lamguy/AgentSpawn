╔══════════════════════════════════════════════════════════════════════════════╗
║                              Color Reference                                 ║
╚══════════════════════════════════════════════════════════════════════════════╝

Header text:     CYAN (semantic: informational)
Tool calls (⏺):  CYAN (semantic: action/invocation)
Tool results (⎿): GRAY/DIMMED (semantic: secondary info)
Error lines:     RED (semantic: error/danger)
Regular text:    DEFAULT (preserves ANSI codes)
Empty states:    GRAY/DIMMED (semantic: muted)
Spinner (⏹):     Shown only for running sessions
